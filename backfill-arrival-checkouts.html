Backfill Arrivals check-out dates

Historical Arrivals rows pre-date the date_mm31c5jt Check-out column (added 2026-05-04). The Make scenarios populate it for new bookings going forward, but existing rows have it blank. This one-shot page sources the check-out from each row's linked Cleans row (the relation column we backfilled in Phase 2) and writes it to Arrivals. Idempotent.

Backfill check-outs

Walks every Arrivals row whose date_mm31c5jt is blank, finds the linked Cleans row(s) via board_relation_mm31mkyp on the Cleans side, reads the most recent non-empty CL_CHECKOUT, and writes it to the Arrivals row.
