Backfill missing bookings

Creates rows on the Cleans board (so the booking shows on rota-planner, laundry rota, hot tub rota) and the Arrivals board (so arrivals-hub and the guest-form pipeline pick it up). Use when a booking didn't make the weekly Guesty report — typically a last-minute direct booking. Sets the Last minute review flag so staff see the row marked as an exception.

Setup

When opened from the deployed site or with the Cloudflare proxy reachable from file:// (allowlist includes null), no token is needed — the proxy attaches its own server-side. Paste a personal Monday token below only if the proxy isn't available.
